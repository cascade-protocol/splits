# @cascade-fyi/cli

Cascade CLI for managing paid MCP services.

## Installation

Download the binary for your platform from [releases](https://github.com/cascade-protocol/splits/releases).

## Usage

```bash
cascade --help
cascade version
```

## Development

```bash
# Run in dev mode
pnpm --filter @cascade-fyi/cli dev

# Build binary
pnpm --filter @cascade-fyi/cli build

# Test binary
./packages/cli/dist/cascade --help
```
